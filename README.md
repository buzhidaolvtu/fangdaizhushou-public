🚀 房贷还款助手 - 息抵月供与资产分析

让你的资产为你打工，用分红瓦解房贷压力。

「房贷还款助手」是一款专为投资者设计的房贷管理工具。它不只是计算月供，更通过直观的 “进度护盾” 视觉设计，帮你实时掌握年度分红对房贷的覆盖率，见证从“还贷人”向“收息人”转变的财务自由进程。

🌟 核心功能

1. 深度房贷模型

组合贷款支持：同时支持商业贷款与公积金贷款。

主流还款方式：完美适配 等额本息 与 等额本金 两种算法。

精确锚点：基于还款起始年份，精准计算任意年份的剩余本金与年度利息支出。

2. 资产分红追踪

资产管理：记录股票、基金等资产的当前市值与原始成本。

预期收益：预设各资产的预期股息率，自动估算全年的被动收入总量。

3. 财务覆盖率分析 (核心特色)

实时进度光环：根据“预期分红 / 年度还款总额”计算覆盖率。

视觉护盾意象：未闭合的圆环象征着正在不断构建的财务保护圈，体现收益覆盖负债的直观进度。

实到流水复盘：详细记录每一笔分红到账，对比“预期”与“实到”，掌握财务实收进度。

4. 未来压力预测

还款曲线：一键预览未来 30 年的月供变化趋势。

动态视角：看清随着贷款本金的递减，你的分红护盾如何一步步变得坚不可摧。
